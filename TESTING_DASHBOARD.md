echo "✅ 令牌就緒: ${ADMIN_TOKEN:0:30}..."

## 📞 快速支援

遇到問題？快速查詢:

| 問題 | 解決方案 |
|------|---------|
| 後端連接失敗 | `cd backend && npm start` |
| 令牌獲取失敗 | 檢查 http://localhost:5000/api/v1/auth/test-token |
| UUID 驗證錯誤 | 使用標準 UUID 格式，參考 API_PATH_FIX.md |
| 403 Forbidden | 確認使用 admin 令牌和 /admin 端點 |
| 響應為空 | 可能無待審核車輛，檢查數據庫 |

**更多幫助**: 查看 TESTING_NEXT_STEPS.md 中的診斷指南部分
